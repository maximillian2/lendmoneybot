# lendmoneybot

Telegram bot to track lending between users using contact sharing Telegram feature.

## Testing locally

```
git clone <url>/lendmoneybot.git
cd lendmoneybot
```

To get things packaged and running simply type

```
mvn package exec:java
```

### Prerequisites

* Java 8 JDK
* Maven 3.0+
